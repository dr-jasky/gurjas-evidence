Literature-neighbour Journal Shortlist

Start with a paper title or 5–10 keywords. The tool finds OpenAlex sources connected to matching literature, then keeps only records identified as journals with an ISSN. It creates a shortlist to investigate—not a submission recommendation.

Journal matching search

Start with a title or keywords

Nothing is sent while you type. When you press Build shortlist — or request sample matching papers for a result — this text is sent directly from your browser to OpenAlex for the lookup; Gurjas does not receive or store it.

Journal-type sources connected to the query

Ranked by matching-work count in OpenAlex after source-type and ISSN checks. This ranking can favour large journals and is not a quality, acceptance-probability or scope-fit score.

Read this before you rely on the list. OpenAlex coverage and source classification can be incomplete. New, niche and regional journals may be under-represented, while high-volume journals may rank highly. Read recent articles and the journal's official scope, then verify identity and current indexing with the Journal Evidence Checker before submitting.

Method and limitations

Method version 0.3-beta · reviewed 20 July 2026. The tool groups matching OpenAlex works by primary source, retrieves metadata for the leading sources, and displays a source only when OpenAlex classifies it as a journal and supplies an ISSN. Matching-work count is a literature-neighbour signal only. For any listed journal you can request up to three sample matching papers, retrieved live from OpenAlex, to inspect why the journal matched.

The query is sent to OpenAlex only after you press the button. Gurjas does not receive it. A displayed source is not endorsed, and absence is not evidence that a journal is unsuitable.
