How the three estimates work

The Fisher-z result concerns detection of a single correlation under the entered effect, power and alpha; it is not a power analysis for the full SEM. The Westland expression uses the entered indicator-to-latent-variable ratio. The N:q result applies a 10:1 ratio to an approximate parameter count. The largest displayed value is shown for comparison only; taking the maximum does not validate any method for a particular model.
