GURJAS WEBSITE - GITHUB PAGES DEPLOYMENT

1. Create a new public GitHub repository: gurjas-evidence
2. Upload every file from this folder into the repository root.
3. Confirm index.html is visible at the root, not inside another folder.
4. Open repository Settings.
5. Open Pages.
6. Select Deploy from a branch.
7. Select branch: main.
8. Select folder: / (root).
9. Save.
10. Wait a few minutes and open the GitHub Pages URL.

After buying a domain, add a CNAME file and update DNS.
