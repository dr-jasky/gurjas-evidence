# Gurjas Evidence and Policy Analytics Website

A static, GitHub Pages-ready website for **Gurjas Evidence and Policy Analytics**.

## Files

- `index.html` — complete one-page website
- `style.css` — responsive premium styling
- `script.js` — navigation, reveal animations, copy-to-clipboard
- `assets/` — logo, favicon, OpenGraph preview
- `404.html` — simple redirect-style error page
- `robots.txt`, `sitemap.xml` — basic SEO placeholders

## Quick GitHub Pages deployment

1. Create a new public GitHub repository, for example `gurjas-evidence`.
2. Upload all files from this folder into the repository root. Keep `index.html` at the root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose branch `main` and folder `/ (root)`.
6. Save and wait a few minutes.
7. Your site will appear at `https://YOUR-USERNAME.github.io/gurjas-evidence/`.

## Recommended custom domain later

- `gurjasevidence.in`
- `gurjasevidence.com`
- `gurjasresearch.com`

When using a custom domain, add a `CNAME` file containing only the domain name.

## Important compliance notes

- Do not publicly display the home-office address unless required.
- Do not claim institutional endorsement unless a formal MoU/approval exists.
- Add Advisory Board names only after written consent and approved wording.
- Keep services framed as methodological, analytical, institutional and policy advisory support, not ghostwriting.
